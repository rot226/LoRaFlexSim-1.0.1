# Expérimentations SNIR Stage 1

Ce répertoire regroupe les ressources (scénarios, données et graphiques) utilisées pour les expérimentations SNIR de la première étape.

## Installation rapide

```bash
python -m venv .venv
source .venv/bin/activate
pip install -e .
```

## Mode FLoRa

Pour calquer les réglages FLoRa (captures inter-SF et seuils de sensibilité), penser à activer `flora_mode=True` lors du lancement des simulations ou de la génération des graphiques.
